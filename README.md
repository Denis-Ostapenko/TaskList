# Task List
## О чём проект:
Список задач. Можно планировать задачи на любой срок.

## Стэк: 
ES6+, React.js 

## Команды проекта:
`npm run start` — запускает сайт для разработки <br>
`npm run deploy` — деплой проекта на gh-pages
